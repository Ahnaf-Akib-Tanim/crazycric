Install git for windows

run command and check whether git is initialized :  git init

then add the github email and username   :  
git config --global user.email "you@example.com"
git config --global user.name "Your Name"

navicate to the project directory and clone the repository: 

git clone https://github.com/Ahnaf-Akib-Tanim/crazycric





During team work: 

If other team members are working on the project, before making new changes, consider pulling any changes from the remote repository to avoid conflicts:   git pull origin master

to save the changes and upload/push to the git:
git add .
git commit -m "Your commit message"
git push origin master
